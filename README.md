# 📝 Basic To-Do List (MEAN Stack)

A simple full-stack To-Do application built using the **MEAN Stack** (MongoDB, Express.js, Angular, Node.js).
This app allows users to add and view to-do items.


## 🧑‍💻 Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/ayaabduljawad750-cyber/Basic-To-Do-List
cd Basic-To-Do-List
```

---

### 2️⃣ Backend Setup

```
cd backend
npm install
```

Run the server:

```
npm start
```

Server runs on:
http://localhost:3000

---

### 3️⃣ Frontend Setup

```
cd frontend
npm install
ng serve
```

Frontend runs on:
http://localhost:4200

---

## 🔗 API Endpoints

| Method | Endpoint | Description    |
| ------ | -------- | -------------- |
| GET    | /todos   | Get all todos  |
| POST   | /todos   | Add a new todo |
